> A fast downloader implements by electron and vue.js

![](preview/01.png)

## Support protocol

* BT .torrent
* Magnet link

#### Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:9080
npm run dev

# build electron application for production
npm run build


# lint all JS/Vue component files in `src/`
npm run lint

```
